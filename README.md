# Supply Teacher Behaviour Scenario

An interactive eLearning scenario created in Articulate Storyline.

## Overview
Learners take the role of a supply teacher starting a lesson while two students at the back continue talking. The scenario asks learners to decide how they would respond and provides feedback on the effectiveness of each approach.

## Learning objective
Help educators reflect on classroom management strategies and consider responses that maintain authority while supporting a positive learning environment.

## Tool used
Articulate Storyline
